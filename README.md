**<u>THE OPEN CALC</u>**

This is an open source ESP-32 powered calculator built for the Earth
Sciences and Technology Science Fair 2026. This project aims to prove
graphing calculators can be produced for a lot less than they are
currently. As well as that, we believe they can also be a lot more
powerful, still for a much cheaper price.

JUST TO BE CLEAR this thing is NOT practical (obviously) as a homemade
calculator isn’t typically allowed in exams. This project is a **proof
of concept.**

Using the power of an ESP-32, this will be able to perform much more
powerful calculations than a TI-85 Plus or CASIO FX9860 G3.

This project is built with:

- ESP-32 S3 16MB Storage (although 8MB storage should do fine as well)

- 3.2 Inch ILI9341 No touch modules

- CardKB v1.1

We might publish our STL case file if we print a case… or not. I don't
care.

This project will aim to have the basic functionality of a graphing
calculator.

This includes:

- Make Graph

- Do Calculate
